Welcome to the Enterprise Mode self-service portal. This portal is intended to help you manage your Enterprise Mode site list requests.

Enterprise Mode is a compatibility mode that lets websites render using a modified browser configuration, helping to avoid many of the common compatibility problems associated with web apps written and tested on older versions of Internet Explorer. Enterprise Mode also helps by redirecting any site that isn’t compatible with Microsoft Edge to the appropriate version of Internet Explorer.

Depending on your role, while you’re here you’ll be able to see the status of your site lists, approve site lists, and push the list into your production environment.

For more info about Enterprise Mode and site lists, see the Enterprise Mode TechNet articles.
